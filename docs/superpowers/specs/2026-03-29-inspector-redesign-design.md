# Inspector 面板美化设计方案

**日期：** 2026-03-29
**状态：** 已批准
**版本：** v2

---

## 1. 概述与目标

对 SmartJourney IDE 右侧 Inspector 面板进行视觉美化，保持简洁轻量的设计风格，同时提升用户体验和信息可读性。

**核心方向：**
- 简洁轻量 — 延续浅色主题，增加微妙视觉层次
- 文件图标 — 为不同文件类型添加直观图标
- Markdown 渲染 — 内容区直接渲染 Markdown 格式

---

## 2. 设计改动详情

### 2.1 文件标签栏布局

**现状：** 垂直标签位于面板最右侧，文字垂直旋转排列，阅读不便

**改动：** 改为横向标签栏，位于标题栏下方

- 标签水平排列，从左到右展示
- 支持横向滚动（当文件数量超过可视区域时）
- 标签样式：`padding: 6px 12px`，圆角 `6px`
- 标签间距：`gap: 6px`

**标签状态：**

| 状态 | 背景 | 边框 | 文字颜色 | 图标颜色 |
|------|------|------|----------|----------|
| 默认 | transparent | transparent | #94a3b8 | #94a3b8 |
| 悬停 | rgba(0,0,0,0.04) | rgba(0,0,0,0.08) | #475569 | #475569 |
| 激活 | #fff | rgba(0,0,0,0.08) | #d97706 | #d97706 |

**激活态额外效果：**
- 添加微妙阴影：`box-shadow: 0 1px 2px rgba(0,0,0,0.04)`
- 文字和图标变为橙色（#d97706）

### 2.2 标题栏

**现状：** 包含"Inspector"标题 + "已同步" + "保存"按钮

**改动：** 简化标题栏，仅保留必要元素

- 左侧：代码图标 + "Inspector" 文字
- 图标颜色：#d97706（橙色）
- 文字样式：font-size 13px，font-weight 600
- **删除：** "已同步" 和 "保存" 按钮

### 2.3 内容区域

**现状：** 纯文本展示，无格式区分

**改动：** Markdown 渲染展示

**渲染样式规则：**

| 元素 | 样式 |
|------|------|
| h1 | font-size 18px, font-weight 700, 底部边框分隔 |
| h2 | font-size 14px, font-weight 600 |
| p | font-size 13px, line-height 1.7, color #475569 |
| strong | color #0f172a, font-weight 600 |
| ul/ol | margin-left 16px, li margin-bottom 4px |
| 引用块 | 背景 #f8fafc, 左侧 3px 橙色边框, 圆角 6px |

**字体：** 继承系统字体栈

### 2.4 底部状态栏

**现状：** 显示实时同步状态

**改动：** 简化显示

- 移除实时同步指示器
- 仅显示：`Markdown · UTF-8`
- 字体：font-size 11px, color #94a3b8

## 3. 组件结构

```
Inspector Panel
├── Header (标题栏)
│   ├── Icon (代码图标)
│   └── Title (文字)
├── Tabs (横向文件标签栏)
│   ├── Tab (每个文件)
│   │   ├── Icon (文件图标)
│   │   └── Label (文件名)
│   └── ... (可滚动)
├── Content (内容区域)
│   └── Markdown Renderer
└── Footer (底部状态栏)
```

---

## 5. 设计原则

1. **少即是多** — 不添加不必要的功能，专注于视觉优化
2. **微妙层次** — 使用 rgba 极淡边框（0.04-0.08）创造分隔，不抢视线
3. **一致体验** — 颜色、圆角、间距与现有设计系统保持一致
4. **功能清晰** — 文件图标帮助快速识别，Markdown 渲染提升可读性
